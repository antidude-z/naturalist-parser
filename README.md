# 🌿 Naturalist Parser

**Naturalist Parser** — это легковесная утилита на Python для массового скачивания изображений с платформы [iNaturalist](https://www.inaturalist.org/).

Скрипт предназначен для быстрого формирования датасетов изображений дикой природы (по умолчанию настроен на снежных барсов, но может быть адаптирован под любые виды) на основе CSV-экспорта наблюдений.

## ✨ Возможности

- 📥 **Массовая загрузка**: Автоматическое скачивание изображений по списку URL.
- 🆔 **Сохранение UUID**: Файлы сохраняются с уникальными идентификаторами наблюдений (UUID), что позволяет легко сопоставлять изображения с метаданными.
- 📊 **Отчетность**: Подробный вывод статистики по успешным и неудачным загрузкам в реальном времени.
- ⚡ **Быстродействие**: Использование `tqdm` для визуализации прогресса и `requests` для эффективной работы с сетью.
- 🛠️ **Простота**: Минимум зависимостей, запуск одной командой через `uv`.

## 📂 Структура проекта

~~~text
naturalist-parser/
├── images/             # (Создается автоматически) Папка для скачанных изображений
├── main.py             # Основной скрипт загрузки
├── observations.csv    # Входной файл со списком наблюдений (UUID и URL)
├── pyproject.toml      # Конфигурация проекта и зависимости
├── uv.lock             # Файл блокировки версий зависимостей
└── .gitignore          # Настройки игнорирования файлов Git
~~~

> [!IMPORTANT]
> Файл `observations.csv` должен содержать как минимум колонки `uuid` и `image_url`. Вы можете заменить его на свой собственный экспорт с iNaturalist.

## 🚀 Установка и запуск

### Предварительные требования

- Установленный [Python](https://www.python.org/) (версия 3.14 или выше, согласно `pyproject.toml`)
- Пакетный менеджер [uv](https://github.com/astral-sh/uv)
- Git

### Пошаговая инструкция

1. **Клонируйте репозиторий:**
   ~~~bash
   git clone https://github.com/Kakoy-Nikakoy-Corp/naturalist-parser.git
   cd naturalist-parser
   ~~~

2. **Установите зависимости:**
   ~~~bash
   uv sync
   ~~~

3. **Подготовьте данные (опционально):**
   Если вы хотите скачать изображения других видов, замените файл `observations.csv` на свой. Убедитесь, что в нем есть столбцы:
   - `uuid`: Уникальный идентификатор наблюдения.
   - `image_url`: Прямая ссылка на изображение.

4. **Запустите скрипт:**
   ~~~bash
   uv run main.py
   ~~~

## 📤 Результат работы

После завершения работы скрипта:
- Все успешно скачанные изображения появятся в папке **`images/`**.
- Имена файлов будут соответствовать UUID наблюдений (например, `a1b2c3d4-e5f6-7890.jpg`).
- В консоли будет выведена сводка:
  ~~~text
  Successful: 150
  Failed: 5
  Total: 155
  ~~~
